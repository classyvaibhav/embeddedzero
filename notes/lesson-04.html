Op-Amps — The Golden Rules

The Swiss Army knife of analog electronics. Two rules solve 80% of circuits. Amplify small signals, buffer high-impedance sources, compare voltages.

By the end of this lesson, you will:
  • Apply the "Golden Rules" to analyse any op-amp circuit
  • Recognise inverting, non-inverting, buffer, and comparator configurations
  • Calculate gain for any of them in 10 seconds

1What's an Op-Amp?

An operational amplifier is an integrated circuit (IC) with a single purpose: amplify the difference between two input voltages by a huge amount (100,000+ times).

It has three pins that matter:

  • V+ (non-inverting input)
  • V− (inverting input)
  • Vout (output)

Plus two supply pins (+Vs, −Vs) that you connect to power. Those are usually omitted from schematics to keep them clean — but they're always needed in real circuits.

Standard op-amp symbol. The triangle points in the direction of signal flow. LM358 is a popular beginner op-amp.

2The Two Golden Rules

When an op-amp is used with negative feedback (output connected back to V−), these two rules are always true:

🏅 Golden Rule 1 V+ = V− — the two input voltages are equal. The op-amp self-adjusts its output until this is true.
🏅 Golden Rule 2 No current flows into either input pin. The input impedance is essentially infinite.

That's it! These two rules let you solve any op-amp circuit in 30 seconds on paper. Let's see them in action.

3The Four Configurations You'll Meet

1. The Voltage Follower (Buffer)

Gain = 1. Seems useless, right? But it's magical: it takes a weak signal (high output impedance, like from a voltage divider) and outputs the same voltage with very low output impedance. Now you can drive a real load without crashing the source voltage.

Vout = Vin

2. The Non-Inverting Amplifier

Amplifies the input. Minimum gain is 1 (so always ≥1).

Vout = Vin × (1 + Rf / Rg)

Example: Rf=10k, Rg=10k → gain = 2. Input 1V → output 2V.

3. The Inverting Amplifier

Amplifies AND inverts (flips sign). Can have gain less than 1 (attenuates).

Vout = −Vin × (Rf / Rin)

Example: Rf=10k, Rin=1k → gain = −10. Input +0.5V → output −5V.

4. The Comparator

Not used with feedback. Output slams to the positive or negative rail depending on which input is higher. Useful for threshold detection.

If V+ > V−: Vout = +Vs
If V+ < V−: Vout = −Vs (or 0V)
The same op-amp symbol in four different circuits. Feedback network determines behaviour.
💻 Try It Live

Build a Non-Inverting Amp (Gain=2) in Tinkercad

Let's physically see what "gain of 2" means. Feed in 1.5V, watch the output be 3V.

Build it:
  1. Open Tinkercad Circuits
  2. Add: 9V battery (for op-amp power), 1.5V battery (signal source), LM358 op-amp IC
  3. Wire LM358 pin 8 (Vcc) to +9V, pin 4 (GND) to ground
  4. Non-inverting input (pin 3) → 1.5V source
  5. Output (pin 1) → 10k R → inverting input (pin 2) → another 10k → GND (this is Rg)
  6. Attach a multimeter between output and ground
  7. Start simulation — meter should read 3V (double the input)
  8. Experiment: change Rf to 20k. What's the new gain? (Answer: 1 + 20k/10k = 3, so output = 4.5V)

4Why Op-Amps Matter for Embedded Work

In an embedded system you'll use op-amps for:

  • Signal conditioning: Scale a 0–1V sensor up to 0–3.3V so your ADC sees a full range.
  • Buffering: Drive a long cable or capacitive load from a high-impedance source.
  • Threshold detection: A comparator turns on an LED when light drops below a level (with an LDR voltage divider).
  • Active filters: Low-pass, high-pass, band-pass audio/sensor filters (advanced).
💡 Which Op-Amp to Buy? For beginners, the LM358 is perfect: dual op-amp (two in one chip), runs on single 5V or 3.3V supply, cheap (~₹15), handles most hobby tasks. When you need higher performance, move to TL072 or MCP6002.

🔍 Check Yourself

Non-inverting amp with Rf=47k and Rg=4.7k. Gain?
Gain = 1 + Rf/Rg = 1 + 47/4.7 = 11.
You want to attenuate a signal to 1/3. Inverting or non-inverting?
Inverting. Non-inverting can only amplify (minimum gain = 1). Inverting can go below 1 (Rf/Rin < 1 means attenuation).
What's a voltage follower useful for?
Converting a high-impedance source into a low-impedance source. Example: a voltage divider's output can't drive a load (loading effect), but pass it through a buffer first and now you can drive a real load without crashing the divider.
Can Vout exceed V+supply?
No. Op-amp output is limited by its supply rails. If you power with +9V, output maxes at about 8V (standard op-amp). Rail-to-rail op-amps can get within ~0.1V of the rails.

Key Takeaways

  • Op-amps amplify the difference between V+ and V− by a huge factor.
  • Golden Rule 1: with negative feedback, V+ = V−.
  • Golden Rule 2: no current flows into inputs.
  • Four common configs: buffer (G=1), non-inverting (G≥1), inverting (G can be anything), comparator.
  • LM358 is the beginner's op-amp — cheap, single-supply, dual package.
  • Op-amp output can't exceed supply rails. Pick a supply that allows the swing you need.
