2Interview Questions You'll Definitely Get

🔍 Common Technical Questions — Practice Answering Out Loud

What's the difference between I2C and SPI?
I2C uses 2 wires (SDA/SCL) shared by all devices, each with a unique address. Slower (100k–1M Hz) but uses less pins. SPI uses 4 wires per slave (MOSI/MISO/SCK/CS), much faster (10–50 MHz) but needs more pins. Pick I2C for sensors, SPI for SD cards and fast peripherals.
What's volatile and when do you use it?
It tells the compiler "this variable can change unexpectedly, always re-read from memory". Used for variables shared between an ISR and the main loop. Without it, the compiler might cache the value and miss changes from the ISR.
How do you pick a current-limiting resistor for an LED?
R = (Vsupply − Vf) / If. For a red LED (Vf=2V) on 5V supply at 20 mA: R = (5−2)/0.02 = 150 Ω. Then check power P = VR×I = 60 mW, so 1/4 W resistor is fine.
Why can't you power an MCU through a voltage divider?
Voltage dividers only work for signal inputs (tiny current). MCUs draw real current, which causes the output voltage to drop dramatically under load. Use a regulator chip instead (LDO or buck).
Walk me through the boot process of an Arduino.
Power stabilises → /RESET released → clock crystal starts → bootloader in Flash runs (checks for new firmware via USB) → if none, jumps to user code → C runtime initialises variables → setup() runs once → loop() runs forever.
Can you explain a project you built?
YOUR ANSWER (practise this!): "I built a multi-sensor environmental monitor using an ESP32. It reads temperature and humidity from a DHT22, and pressure from a BMP280 sensor — all via I2C. The data displays on a 0.96-inch OLED. A button lets me cycle between summary and detailed views, and I used an interrupt with debouncing for the button handling. The full BOM came to around 1,300 rupees. The code is on GitHub with a README and wiring diagram."
What's the flyback diode for?
When switching off an inductive load (relay coil, motor), the collapsing magnetic field generates a large reverse voltage spike. The flyback diode (connected in reverse across the load) absorbs that spike safely, protecting the transistor from destruction.
Why do we put 100 nF caps on IC power pins?
It's a decoupling (bypass) capacitor. At high switching frequencies, it acts as a local charge reservoir that shorts noise to ground before it propagates through the board. Every IC needs one right next to its Vcc pin — no exceptions.

4Where to Continue Learning (After Your First Job)

Going Deeper into Embedded

  • STM32 with HAL — STM32 is the industry standard for mid-range embedded. Arduino is training wheels; STM32 is the bike.
  • FreeRTOS — Real-time operating system. Learn it once you're comfortable with bare-metal.
  • Bare-metal C on ARM Cortex-M — When you really want to understand what's happening below Arduino.
  • PCB design in KiCad / Altium — From schematic to manufactured board. Hugely valuable skill.

Going Deeper into Linux Embedded

  • Raspberry Pi projects — First step into Linux-based embedded.
  • Yocto Project — Building custom embedded Linux distributions (this is where Vaibhav works at Omnidya).
  • Device drivers — Writing kernel-level code for hardware.
  • NXP i.MX, Rockchip, Qualcomm platforms — industrial-grade SoCs.
