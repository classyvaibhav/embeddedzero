C.4

Arduino / ESP32 Code Snippets

Quick ref

Non-blocking interval

unsigned long last = 0;
const unsigned long INTERVAL = 1000;
void loop() {
  if (millis() - last >= INTERVAL) {
    last = millis();
    // periodic thing
  }
}

Debounced button

const int BTN = 4;
unsigned long lastChg = 0;
int lastState = HIGH;

bool btnPressed() {
  int s = digitalRead(BTN);
  if (s != lastState && millis() - lastChg > 50) {
    lastChg = millis(); lastState = s;
    return s == LOW;
  }
  return false;
}

I2C scan

#include <Wire.h>
void setup() {
  Serial.begin(115200); Wire.begin();
  for (byte a=1; a<127; a++) {
    Wire.beginTransmission(a);
    if (Wire.endTransmission()==0) Serial.printf("0x%02X\n", a);
  }
}

C.6

Mnemonics & Quick Reminders

Quick ref
  • "B. B. Roy of Great Britain has a Very Good Wife" — colour code mnemonic (Black, Brown, Red, Orange, Yellow, Green, Blue, Violet, Grey, White = 0–9).
  • "NPN — Never Points iN" — arrow on NPN emitter points outward.
  • "MOSI = Master Out Slave In; MISO = Master In Slave Out"
  • "Current flows through, voltage sits across, power dissipates" — language precision.
  • Ohm's wheel: V=IR, I=V/R, R=V/I, P=VI=I²R=V²/R.
  • 5 time-constants rule: cap practically full / empty after 5τ.
  • Decoupling rule: 100 nF ceramic near every Vcc pin. No exceptions.
  • Pull-up default: 10 kΩ for GPIO.
  • I2C pull-ups: 4.7 kΩ default; 2.2 kΩ at higher speeds.
  • ADC averaging: ESP32 noisy — average 10 reads.
  • Flyback diode: always across inductive loads.
  • Cap voltage margin: rating ≥ 2× expected.
  • SMD resistor code: "103" = 10 kΩ; "4R7" = 4.7 Ω.
  • Absolute max ≠ operating max — it's the destruction threshold.
